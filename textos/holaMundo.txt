# Prueba de Bloques Duales

## Bloque RUN (Contexto: Raíz del proyecto)
Este comando lista los archivos de la raíz usando el `workingDir`:
```run
ls -F
```

## Bloque SCRIPT (Contexto: Carpeta de scripts)
Este comando ejecuta un script dentro de `scriptsDir`:
```script
node holaMundo.js
```

¡Ambos contextos funcionando por separado!
