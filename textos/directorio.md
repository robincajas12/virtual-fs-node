# Explorador de Proyecto

## Archivos en la Raíz
Listado obtenido mediante `run`:
```run
ls -p | grep -v /
```

## Scripts Disponibles
Listado obtenido mediante `script` (dentro de la carpeta de scripts):
```script
ls *.js
```

## Información de Red
Un comando rápido de sistema:
```run
hostname
```
